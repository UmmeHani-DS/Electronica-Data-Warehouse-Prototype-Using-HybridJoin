#STEPS TO RUN THE HYBRIDJOIN PROJECT

1) Firslty download and extract the project zip.

2) The folder will contain createDW.sql file. Open mysql and open the createDW file
then run it.

3)Import the extracted dwh_project folder into eclispse. Make sure you add the 
provided jar files. You can add the jar files by right clicking on the project
then clicking properties. After that click on the java build path their you will
see the section of libraries, Their click add external jars and apply then close.

4) run the project in the hybridjoin file. The prompt will ask for url, username
and password provide those and the hybridjoin will start.

5) I have also provided the query file you can run that to see the outcome.
